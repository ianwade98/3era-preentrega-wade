Camila Malumbres

Escritora | Autora "Los que no piden permiso"

Para llorar. Para doler. Para sanar.

Para seguir. Para parar. Para confiar.

Para arriesgar. Para sentir. Para reír.

Para frustrar. Para desear. Para caer.

Para volver. Para flotar.

Para vivir. Para morir. Para nacer.

Para elegir. Para contar. Para creer.

Para eso sirve escribir.

Para que el vacío se convierta en algo y

ese algo en aire y ese aire en lo único

para vivir.

Puede que alguno de estos escritos haya sido parido en un acto de desesperación y

puede que otros provengan de una puja diferente a la mía. En ambos casos confirmo

lo sospechado, el dolor es inherente a la vida. Elegí escribir un libro, para llorar.

Elegí publicarlo, para que llore alguien más.

El llanto es otra forma de volver a nacer. La escritura no lo es menos.
